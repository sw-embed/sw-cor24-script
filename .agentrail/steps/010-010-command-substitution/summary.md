Implemented (cmd args...) parenthesized command substitution and arithmetic commands (+, -, *, /, %). Command substitution uses an innermost-first iterative strategy in resolve_cmdsub() that processes the raw line string before tokenization, with a function pointer indirection to break mutual recursion with eval_line. Skips parens inside quoted strings and brace blocks. Added 16 new tests (93 total, all passing). Committed and pushed as 3c2168c.
